# Baiswara Golden Express

A full-stack bus booking application built with Spring Boot and Angular.

## Structure
- `backend/`: Spring Boot Application (Java 17, Maven)
- `frontend/`: Angular Application

## Prerequisites
- Java JDK 17+
- Node.js & npm
- MySQL Server

## Setup Instructions

### Backend
1. Navigate to `backend/`
2. Update `src/main/resources/application.properties` with your MySQL credentials.
3. Run the application:
   ```bash
   mvn spring-boot:run
   ```
   *The server will start on http://localhost:8080*

### Frontend
1. Navigate to `frontend/`
2. Install dependencies (simulated):
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   ng serve
   ```
   *Open http://localhost:4200 in your browser*

## Features
- **Search**: Find buses by city and date.
- **Booking**: Select seats visually and book tickets.
- **Auth**: JWT based Login/Register.
- **Admin**: Dashboard to view stats and manage routes. (Access: Login as Admin)

## Tech Stack
- **Frontend**: Angular, Angular Material
- **Backend**: Spring Boot, Spring Security (JWT), Spring Data JPA
- **Database**: MySQL
